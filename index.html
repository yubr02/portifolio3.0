Login Seguro JWT

Sistema desenvolvido para reforçar a segurança de acesso em aplicações web, com autenticação mais confiável e melhor controle de sessão.

Problema

Sistemas web precisam controlar acesso de forma segura, evitando autenticações frágeis e fluxos confusos para o usuário.

Arquitetura

Estruturei o projeto com API REST em Golang, camada de autenticação com JWT, front-end em React e persistência de dados em MySQL.

Solucao

Implementei um fluxo de login com validação de credenciais, emissão de token e controle de sessão para proteger rotas e melhorar a segurança da aplicação.

Decisões técnicas

Usei JWT pela praticidade no controle de sessão, Golang pela performance e simplicidade da API, e React para deixar a experiência de autenticação mais fluida.
